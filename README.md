# SpaceInVaders - Projet IUT Lens

Ce projet a été réalisé par Olivier Dourdin dans le cadre de son cursus à l'IUT de Lens.

Le SpaceInVaders est un jeu de type shoot'em up, dans lequel le joueur doit détruire des vagues d'ennemis.

## Fonctionnalités

Dans ce jeu, le joueur peut :
- Se déplacer à l'aide des touches directionnelles
- Tirer des projectiles à l'aide de la barre espace
- Gagner des points en détruisant des ennemis
- Perdre des vies en se faisant toucher par un ennemi
- Perdre la partie si le nombre de vies tombe à 0
- Gagner la partie si le joueur détruit tous les ennemis
